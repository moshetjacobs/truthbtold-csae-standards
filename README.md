# TruthBTold.Space – CSAE Standards

This repository contains the publicly accessible Child Sexual Abuse and Exploitation (CSAE) Standards for **TruthBTold.Space**.

Google Play requires social and communication apps to publish external standards describing their policies, enforcement practices, and reporting procedures related to CSAE. This repository hosts those standards via GitHub Pages.

- `index.md` – The full CSAE Standards document displayed on GitHub Pages.
